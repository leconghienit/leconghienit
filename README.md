## 🚀 What I Am Building

### 🍔 Food Delivery Backend

A backend-focused system built to practise real-world architecture:

- NestJS microservices with BFF, authentication and user services
- TCP/gRPC communication between services
- JWT access token and refresh token authentication
- Centralized exception handling, logging and request tracing
- MySQL/PostgreSQL with Prisma ORM
- Nx monorepo and pnpm workspace

### 🐾 Pet Shop Platform

A full-stack e-commerce project with:

- Product search, filtering, sorting and pagination
- Cart, checkout, order management and admin dashboard
- JWT authentication and email OTP verification
- React, Redux Toolkit, Node.js, Express and MongoDB

### 💬 Realtime Chat Application

A realtime messaging project using:

- Socket.IO communication
- Conversation and message persistence
- One-to-one chat rooms
- Realtime notifications and online status concepts
